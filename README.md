### Recharge Subscription Sync Service For Healthy Baby Store

This project is a custom webhook handler hosted on **Vercel** that automates the update of **Recharge subscription properties**. 

The workflow is managed via this **GitHub** repository and was tested/configured using **Postman**. Whenever a charge occurs, the script dynamically fetches the current product data and updates the subscription's metadata to ensure the original price and discount details are always accurate and synchronized between Shopify and Recharge.
